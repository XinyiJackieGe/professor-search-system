Software

  • Grandet, a unified, economical object store for web applications. The scheme is implemented as described in the SoCC’16 paper.

  • AppDoctor, a tool for efficiently and effectively detecting mobile app bugs. The scheme is implemented as described in the EuroSys’14 paper.

  • NeonGoby, an alias analysis checker. The scheme is implemented as described in the FSE’13 paper.

  • FADE, a secure cloud storage with file assured deletion. The scheme is implemented as described in the SecureComm’10 paper.

  • Welly, a Cocoa application on Mac OS X, bringing best user experience for browsing term BBS.

  • DynamoRIO, a dynamic instrumentation tool platform.
